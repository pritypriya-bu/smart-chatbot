# Screenshots

Drop your app screenshots here with these filenames so the README picks them up:

- `screenshot_main.png` — the main chat interface
- `screenshot_data_grid.png` — the data grid with a loaded CSV/Excel
- `screenshot_code_fix.png` — a code fix showing the diff
- `screenshot_rag.png` — the knowledge-base sidebar with docs loaded and a source-cited answer

Suggested size: 1600x900 (or any 16:9), PNG, under 500 KB each.

You can capture them with:
- **Windows:** Win + Shift + S
- **macOS:** Cmd + Shift + 4
- **Linux:** GNOME Screenshot / `flameshot`
